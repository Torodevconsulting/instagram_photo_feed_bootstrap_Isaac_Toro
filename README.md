📸 Instagram Feed — Bootstrap Template
A responsive Instagram-style profile feed built with Bootstrap 5, Bootstrap Icons, and vanilla JavaScript.
Features

Grid view — 3×3 photo grid with hover overlay showing likes and comments
List view — card-based feed with caption, date, timestamp and stats
Toggle between grid and list view with active state indicator
Sticky navbar with "Create a New Post" button and settings dropdown (Profile, Accessibility, Privacy and Data, Log Out)
Modal to create new posts with caption textarea, photo upload and location buttons
Fully responsive — optimized for mobile, tablet and desktop

Built with

Bootstrap 5.3
Bootstrap Icons 1.11
HTML5 + CSS3
Vanilla JavaScript

What I learned

Bootstrap grid system and responsive utilities (col-4, d-none d-md-inline)
Component usage: modals, dropdowns, sticky navbar
CSS aspect-ratio, object-fit and max-height for image control
DOM manipulation with getElementById and classList.toggle
Debugging common issues: id vs class, min-height vs max-height, inline styles overriding CSS
